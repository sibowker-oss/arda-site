# Arda — Marketing Site UI Kit

A rebuild of `ardacomputing.com` applying the locked **Arda Tone of Voice v1** and the visual foundations.

> ⚠️ **This is a rebuild proposal, not a recreation.** The current live site (May 2026) is off-voice — see the ToV doc §10 "Before / After web copy". This kit shows what an on-voice marketing site looks like *with the design system applied*. Treat it as a forward-looking reference, not a clone of what's currently shipped.

## Components

- `Header.jsx` — top nav with ARDA mark, primary nav, dual CTAs.
- `Hero.jsx` — homepage hero (sober, sovereign, specific). Lattice texture in the background.
- `OwnOperateGovern.jsx` — the structural three-column claim layout (the ToV's owned device).
- `BoundaryDiagram.jsx` — the "inside your trust boundary" architecture frame.
- `ThreeBarriers.jsx` — Access / Security / Cost · Challenge / Opportunity / Arda's Answer table.
- `StatsBand.jsx` — large numerals on dark, with footnoted sources.
- `CTASection.jsx` — book-a-deep-dive close.
- `Footer.jsx` — sober footer, Australian English, status line.

## Screens (single-page route)

`index.html` mounts the full marketing page top-to-bottom. Click anchor links in the header to jump.
